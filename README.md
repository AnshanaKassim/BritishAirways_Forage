# BritishAirways_Forage
This repository contains two projects completed as part of the British Airways Data Science virtual experience program. The work focuses on applying data analysis and machine learning to real-world airline business problems.

## 📂 Projects

### 🔹 Task 1: Lounge Demand Modelling (Excel)
- Built a scalable lookup table to estimate lounge eligibility across flight categories 
- Engineered features and grouped flights by haul type, time of day, and region 
- Used pivot tables to derive data-driven eligibility percentages 
- Enabled flexible forecasting of lounge demand for future schedules


### 🔹 Task 2: Customer Booking Prediction (Machine Learning)
- Developed a Random Forest model to predict booking behaviour
- Addressed class imbalance using balanced weighting
- Improved recall from 13% → 62% using threshold tuning
- Identified key drivers of booking using feature importance

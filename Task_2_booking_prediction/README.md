# Task 2: Customer Booking Prediction 

### 🎯 Objective:
Predict whether customers will complete a booking before travel. 

### ⚙️ Approach:
- Data cleaning and feature engineering 
- Random Forest model with balanced class weighting 
- Threshold tuning to improve recall 
- Cross-validation for reliability 

### 📊 Model Performance:
- Accuracy: 76%
- Precision: 34% 
- Recall: 62% 
- F1 Score: 0.44 

### 🔍 Key Drivers:
- Purchase lead time is the most important factor 
- Trip characteristics influence booking behaviour 

### 💡 Key Insight: 
- Recall improved from 13% → 62% through threshold tuning. 

### 💼 Business Impact:
- Enables early identification of high-intent customers 
- Improves marketing targeting 
- Increases conversion rates
